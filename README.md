# Yoke Thé — The Art of Myanmar Marionette

A modern cultural storytelling website showcasing **Myanmar’s traditional marionette theatre (Yoke Thé)**.

This project presents the **28 sacred puppet characters**, their roles in the cosmic order, and legendary puppeteers — all through an elegant UI built using **pure HTML and CSS**.

---

## Features

* Traditional + Modern cultural design
* Complete **28-character system**
* Interactive modal-based storytelling
* Smooth hover animations and transitions
* Structured image layout (consistent UI)
* Built with **HTML5 & CSS3 only** (no JavaScript)

---

## Character System

The 28 figures are structured into six groups:

* **Animal Kingdom** — cosmic beginnings (Horse, Tiger, Elephant, etc.)
* **Mythical Beings** — power and conflict (Zawgyi, Naga, Garuda, Ogres)
* **Higher Realms** — divine entities (Nat-Pyet, Deva, Brahma)
* **Royal Court** — governance and story core (King, Prince, Ministers)
* **Humans & Sages** — wisdom and morality (Hermit, Brahmin, Clowns)
* **Commoners** — everyday life (Old Man, Old Woman, Extra)
